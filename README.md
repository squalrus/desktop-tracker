# Windows Virtual Desktop Time Tracker

A lightweight, local productivity tool that monitors how much time you spend on each Windows Virtual Desktop. It quietly logs your usage in the background and visualizes your daily statistics on a clean, responsive web dashboard. 

This is perfect for freelancers, remote workers, or anyone looking to separate their "Work" desktop context from their "Personal" desktop context natively within Windows.

## ✨ Features

* **Automatic Tracking:** Seamlessly records the time spent on your currently active virtual desktop.
* **Smart Pausing:** Automatically stops tracking when your Windows machine is locked, ensuring accurate data.
* **Privacy First:** All data is stored locally in a simple `desktop_data.json` file. No cloud syncing, no accounts.
* **Real-time Dashboard:** A local web interface that updates automatically every 10 seconds.
* **Detailed Breakdown:** Displays total time, individual desktop time (in hours, minutes, and seconds), and visual percentage bars.
* **Automatic Dark Mode:** The dashboard instantly matches your system or browser's Light/Dark mode preference.

---

## 📋 Prerequisites

To run this application, you will need:
* **Operating System:** Windows 10 or Windows 11
* **Python:** Python 3.x installed (ensure Python is added to your system PATH during installation).
